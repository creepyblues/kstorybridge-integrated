# Welcome to KStoryBridge Components

KStoryBridge's shared component library provides a consistent design system across our monorepo applications. Built on shadcn/ui and Radix UI primitives, these components ensure consistency between our Dashboard, Website, and Admin applications.

<div className="subheading">Design System</div>

Our component library is built with:
- **shadcn/ui** - High-quality, accessible components
- **Radix UI** - Unstyled, accessible primitives
- **Tailwind CSS** - Utility-first styling
- **TypeScript** - Type-safe component development

<div className="subheading">Getting Started</div>

To use KStoryBridge components in your application:

```bash
# Install the shared UI package
npm install @kstorybridge/ui

# Import components in your React app
import { Button, Card, Input } from '@kstorybridge/ui';
```

<div className="subheading">Component Categories</div>

- **Form Elements** - Button, Input, Select, Checkbox, etc.
- **Layout** - Card, Separator, Tabs, etc.  
- **Feedback** - Toast, Alert, Progress, etc.
- **Navigation** - Dropdown Menu, Dialog, etc.
- **Data Display** - Badge, Skeleton, Table, etc.
